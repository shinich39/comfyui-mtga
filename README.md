# comfyui-mtga

Make Textarea Great Again

## Usage  

- Ctrl+/: Commentify
- Tab, Shirt+Tab: Indentify
- {, (: Auto-Close brackets
- Ctrl+Enter, Ctrl+Shift+Enter: Line break
- Start with "@": Artists
- Start with "#": Characters
- Start with any characters: All tags
